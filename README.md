# Interruptor Game

Juego web del impostor en espanol, pensado para jugar en modo `pass & play` desde un solo dispositivo.

## Objetivo

Crear una experiencia local por turnos donde:

- un jugador es el impostor,
- el resto conoce una palabra secreta,
- cada participante da una pista,
- todos votan,
- y el impostor tiene una ultima oportunidad para adivinar la palabra si lo descubren.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Scripts

```bash
npm install
npm run dev
```
